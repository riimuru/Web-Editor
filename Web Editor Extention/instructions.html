Welcome to Web Editor. My name is Marouane (creater of Web Editor) and I will walk you through, how to use Web Editor and get your first script running. First of all I recommend editing your scripts in a different code editor, since I haven't yet made a saving feature for the editor in the add-code section.

Writing your first script:

If you haven't programmed before I recommend visiting https://www.w3schools.com/js/default.asp and learn with w3schools tutorials AND you can watch tutotial on youtube too they really help alot. The programming language is called JavaScript and will run on webpages immediatly after it has loaded. The code will run on just like code being inserted into the console. 

Instructions:

  1. Go to the add code section.
  2. Choose the URL-matching. The code will run on pages where the URL of the webpage equals or/and contains the URL-match.
  3. Paste or (write) your JavaScript code into the "your code" section.
  4. Click add code.
  5. Now the code will run on webpage matching with the URL-match.
  6. If your code contains an error you will be able to see the error message in the console (press F12) of websites that matches with the URL-match.
